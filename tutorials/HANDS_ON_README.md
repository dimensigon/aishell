## Quick Reference

### Common Commands

```bash
# Start AI-Shell
ai-shell

# Connect to database
ai-shell --db postgresql://user:pass@host:port/dbname

# Start with config file
ai-shell --config config.yaml

# Start GraphQL API
ai-shell --api --port 8080

# Start Web UI
ai-shell --web --port 3000

# Interactive tutorial
./interactive_tutorial.py
```
